OKHTTP的http问题

kotlin中的okhttp默认是不支持http的，所以是要修改

添加该文件，以便可以申请http

文件内容

AndroidManifest.xml中按照位置添加

android:networkSecurityConfig=”@xml/network_security_config”

该文件下添加

implementation ‘com.squareup.okhttp3:okhttp:3.12.1’

注：千万不要升级

MYSQL

mysql的密码不符合要求，ERROR 1819 (HY000): Your password does not satisfy the current policy requirements”

这个与验证密码策略validate_password_policy的值有关。默认是1，所以刚开始设置的密码必须符合长度，且必须含有数字，小写或大写字母，特殊字符。如果不想设置8位，或者想设置简单点，可以选择Policy0.
validate_password_policy有以下取值：

查看当前密码安全策略

查看具体的设置项

修改validate_password_policy参数的值（等级为0）

validate_password_length(密码长度)参数默认为8，我们修改为1
